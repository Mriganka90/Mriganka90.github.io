Featured Project
ABB 800xA Batch Reporter
// pharmaceutical_batch_reporting · opc_da · opc_hda · vba · excel
Problem
A pharmaceutical plant running ABB 800xA had no automated way to generate batch documentation. Engineers were manually pulling OPC tag values, copying trend data, and formatting reports — a process that took 25–30 minutes per batch and introduced transcription errors that could fail GMP audits.
Solution
Built a VBA-driven Excel tool that dynamically reads 23 configurable OPC DA parameter paths, retrieves historical trends via OPC HDA, extracts phase-wise event lists with user tracking, auto-corrects date format locale mismatches in ABB API calls, and outputs a formatted batch report in one click.
Results
Report generation time dropped from ~30 minutes to under 2 minutes — a 93% reduction. Zero manual data entry, eliminating transcription errors. Tool runs inside ABB 800xA's own Excel environment with no external dependencies.
